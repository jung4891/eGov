4. 화면코딩

- 이클립스에서 부트스트랩을 이용해서 html을 만들도록 하겠습니다.

1) 이클립스 실행

2) 프로젝트에 부트스트랩 설치

- 부트스트랩 설치는 지나번에 압축을 푼 폴더를 webapp 밑에 넣으면 됩니다.

- html 디자인의 경로는 /WEB_INF/jsp 밑으로 하겠습니다.

3) 메인화면 코딩

<!DOCTYPE html>

<html lang="ko">

<head>

<title>Bootstrap Example</title>

<meta charset="utf-8">

<meta name="viewport" content="width=device-width, initial-scale=1">

<link rel="stylesheet" href="/css/bootstrap/css/bootstrap.min.css">

<script src="/js/jquery.min.js"></script>

<script src=/css/bootstrap/js/bootstrap.min.js"></script>

</head>

<body>

<div class="container">

<h1>My First Bootstrap Page</h1>

<p>This is some text.</p>

</div>

</body>

</html>

- 모든 화면의 기본 디자인은 부트스트랩 pannel 기능을 이용하겠습니다.

- 메인화면 폴더를 만듭니다.

- board 폴더 생성

- 메인화면 html을 만듭니다.

- html은 html5 기준으로 작업합니다.

- board/list.jsp

- 메인화면에는 로고, 로그인, 로그아웃, 게시판목록이 있습니다.

- 로고를 만듭니다.(생략)

- 로그인 디자인을 합니다.

- 로그아웃 버튼을 만듭니다.

- 게시판목록을 만들어 보겠습니다.

- 검색조건을 만듭니다.(제목, 타이틀)

- 목록 디자인을 합니다.

- 등록버튼을 만듭니다.

- 부트스트랩 예제에서 패널 소스를 복사해서 붙여넣습니다.

​

4) 등록/수정화면 코딩

- 목록에서 등록버튼을 클릭하면 이동합니다.

- 등록화면에는 게시판 입력요소를 폼형식에 넣는 작업을 합니다.

- 버튼은 저장, 삭제 및 취소를 두겠습니다.

5) 상세화면 코딩

- 상세화면은 목록에서 글을 선택했을때 이동합니다.

- 상세화면은 모든 데이터를 뿌려주기만 하면 됩니다.

- 버튼은 수정 및 목록버튼을 두겠습니다.

6) 부트스트랩 예제는 http://bootstrapk.com/components/ 및

https://www.w3schools.com/bootstrap/default.asp w3school에도 있습니다.

7) 이제 html을 눌러서 화면들이 제대로 보이는지 테스트해 봅니다.

화면이 깨지거나 이동이 잘 안되면 될때까지 수정해 주세요..
