2. 프로젝트 생성

====================

** Maven 프로젝트이기 때문에 pox.xml이 생성이 된다.
** java폴더는 java소스가 들어간다.
​** resources폴더는 각종 설정, 서버 설정, 디비 설정, 쿼리가 들어감.
** webapp폴더는 jsp가 들어가는 폴더

** resources > db에 있는 sampledb.sql은 메모리 db로 WAS서버가 올라갈때 자동적으로 테이블생성과 테이터 입력이 되게 된다.
   그래서 서버 껐다 키면 초기화가 된다.

====================

- 이클립스 실행

- 메뉴 > eGovFrame > Start > New Web Project 클릭(메뉴가 안보이면 Window > Open Perspective 에서 eGovFrame 선택)

- 프로젝트명 기입(아무거나) 저는 board 이라고 할께요

- Group Id 도 동일이름 기입 후 next 클릭

- Generate Example 체크 후 Finish 클릭

- 잠시 후 프로젝트가 생성되며 샘플코드가 자동으로 만들어 집니다.




3. 프로젝트 실행

====================

** 전자정부에서는 Controller, VO, Service, ServiceImpl, Mapper(DAO)라 한 세트다.

** 처음에 화면띄우게 되면 webapp의 index.jsp가 실행된다. 그러면 egovSampleList.do가 호출된다.
   > 그럼 컨트롤러를 타게 되고   (egovframework.example.sample.web)
   > 서비스를 타고   (egovframework.example.sample.service)
   > 서비스impl을 탄다.  (egovframework.example.sample.service.impl)
   > 그리고 DAO를 타고 DAO에서 sql(resources/egovframework/sqlmap/example/sample/샘플sql)을 타고
   > 결과가 반환되어 다시 impl에 들어오고 > 그 결과가 서비스로 > 컨트롤러를 갔다가
   > jsp로 돌려져서 사용자에게 보여지게 된다. (webapp/WEB-INF/jsp/egovframework/example/sample) 

** SampleVO는 전자정부에서 VO는 테이블이라고 생각하면 된다.

====================

- 프로젝트는 톰캣7 버전으로 실행해 보도록 하겠습니다.

- 톰캣은 was(웹어플리케이션 서버)라고 하며 web서버 역할도 같이합니다.

- 우리가 만든 시스템을 실행시켜주는 역할을 합니다.

- 먼저 이클립스 화면내에 Servers 탭을 찾으세요. 화면 하단에 보통 있습니다.

- Servers탭에서 마우스 오른쪽 클릭 > New > Server

- Tomcat v7.0 Server 선택 후 Next 클릭

- Download and Intall... 버튼 클릭

- 톰캣을 설치할 폴더를 지정합니다. 새폴더를 만들어서 설치하는게 좋습니다.

- 저는 E:\eGovFrameDev-3.7.0-64bit\eclipse\ 기존 폴더밑에 tomcat7 폴더를 만들어서 설치하였습니다.

- 실행할 프로젝트 선택 후 Finish

- 이렇게 하면 Servers탭에 톰캣정보가 생성됩니다.

- 생성된 톰캣정보를 클릭하고 실행버튼을 클릭하면 톰캣기본포트인 8080으로 실행됩니다.

- Console 로그에 오류없이 실행이 완료되면 익스플로러에서 확인할 수 있습니다.

- http://localhost:8080/board 이렇게 치고 들어가시면 샘플화면을 볼 수 있습니다.

- 화면이 안보이거나 오류가 나신 분들은 댓글을 남겨주세요.
